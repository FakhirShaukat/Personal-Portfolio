# Personal Portfolio Website

A modern and responsive developer portfolio website built using React.js and Tailwind CSS, showcasing my projects, skills, and experience as a Frontend Developer.

## Live Preview

View Live Project: [Add your deployed link here]

## Features

* Modern and clean UI design
* Fully responsive (mobile, tablet, desktop)
* Smooth scrolling navigation
* Interactive project cards with hover effects
* Skills slider with animation
* Download Resume functionality
* Contact section with call-to-action

## Tech Stack

* Frontend: React.js
* Styling: Tailwind CSS
* Icons: React Icons
* Animations: CSS / Tailwind
